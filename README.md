# 關於\[自戀筆記\]

作者是一位精神科醫師，從事臨床工作多年後，想要將心理的專業內容，以較能為一般人理解的方式呈現。成立FB 的專頁「納西斯花園」，並在區塊鍊平台 Matters 上，以「寓森」的名稱發表文章。

這本書是作者研究「自戀」這個主題的筆記，分享給所有關心人類心靈成長議題的人。本書分為四大部份，後面會陸續更新文章。
